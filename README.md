# Crypto-Website

A clean, responsive single-page Crypto Website built using HTML, CSS and JavaScript.

![Screenshot](https://raw.githubusercontent.com/BinaryVortex/Crypto-Website/main/Screenshot%202024-09-10%20123835.png)

Overview

This repository contains a simple cryptocurrency-themed website that demonstrates layout, styling, and small JavaScript interactions. It is ideal as a portfolio project or a starting point for learning front-end development.

Features

- Responsive layout built with HTML & CSS
- Interactive elements powered by vanilla JavaScript
- Clean, modern design suitable for crypto-related content or landing pages

Demo / Running Locally

1. Clone the repository:

   git clone https://github.com/BinaryVortex/Crypto-Website.git

2. Open the project folder and open `index.html` in your browser (double-click or use a local server).

   Example with a simple Python server:

   ```bash
   cd Crypto-Website
   python3 -m http.server 8000
   # then open http://localhost:8000 in your browser
   ```
